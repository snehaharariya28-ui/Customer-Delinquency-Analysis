# Customer Delinquency Analysis

## Objective
Analyze customer data to identify factors influencing delinquency.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis
- Behavioral Analysis (Missed Payments)
- Feature Engineering (Risk Category)

## Key Insights
- No single feature strongly predicts delinquency
- High overlap between delinquent and non-delinquent customers
- Delinquency is a multi-factor problem

## Files
- analysis.py → main code
- project_details_report.pdf → detailed report

## Conclusion
Customer delinquency depends on multiple factors rather than a single variable.
